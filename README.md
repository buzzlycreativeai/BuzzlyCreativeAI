BuzzlyCreative AI - Next.js (App Router) starter (Dark Moderno)
----------------------------------------------------------------
This starter contains:
- Dark landing with aurora effect (App Router)
- Central chat UI calling /api/chat
- Server API route that proxies to OpenAI if OPENAI_API_KEY is set, otherwise returns a simulated response
- .env.example for environment variables

Quick start:
1. npm install
2. npm run dev
3. Open http://localhost:3000

Deploy: push to GitHub and import into Vercel.
